# Payroll Management System

A desktop-based Payroll Management System developed using Java Swing, JDBC, and MySQL following the MVC (Model-View-Controller) architecture.

## Features

- Admin and Employee Login
- Employee Management (Add, Update, Delete)
- Attendance Management
- Payroll Calculation
- Payslip Generation
- MySQL Database Integration
- Input Validation and Error Handling

## Technologies Used

- Java
- Java Swing
- JDBC
- MySQL
- NetBeans IDE
- MVC Architecture

## Project Structure

```
src/
test/
nbproject/
```

## How to Run

1. Open the project in NetBeans.
2. Configure the MySQL database connection in the source code.
3. Run the project.
