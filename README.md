# Payroll Management System

A desktop-based Payroll Management System developed using Java Swing, JDBC, and MySQL following the MVC (Model-View-Controller) architecture.

## Features

- Secure Admin and Employee Login
- Employee Management (Add, Update, Delete)
- Attendance Management
- Automated Payroll Calculation
- Payslip Generation
- MySQL Database Integration
- Input Validation and Error Handling

## Technologies Used

- Java
- Java Swing
- JDBC
- MySQL
- NetBeans IDE
- MVC Architecture

## Project Structure

```
src/
test/
database/
screenshots/
```

## Database Setup

1. Open MySQL.
2. Create a database named `payroll_system` (or run the SQL script).
3. Import the `database/payroll.sql` file.
4. Update the database username and password in the project if required.

## How to Run

1. Open the project in NetBeans.
2. Configure the MySQL database connection.
3. Run the project.
